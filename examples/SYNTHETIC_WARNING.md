# Synthetic Data Warning

**All data in this folder is entirely synthetic and fabricated for demonstration purposes.**

- No real user data, session tokens, or personally identifiable information (PII) is included
- Domain names and endpoints reference real platforms but all request/response content is synthetic
- Numbers, ratios, and statistics are illustrative and do not represent actual measurements
- These examples exist solely to show the output format of each tool

If you want real results, capture your own HAR file following the instructions in [`capture/CAPTURE_GUIDE.md`](../capture/CAPTURE_GUIDE.md) and run the analysis tools against it.
